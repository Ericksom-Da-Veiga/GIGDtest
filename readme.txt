Projeto escolar realizado em Python, este projeto tem o intuito de usar as imagens de caras de pessoas fornecidas pelo nosso banco de imagens(pasta img) e comparar com as pessoas que estão a ser observadas pela ‘webcam’ do computador, também podem ser configurados outras fontes de capturas de imagens(câmaras), no caso de face ser reconhecida vai aparecer o nome da pessoa em baixo do rosto da mesma.

Para inicializar este projeto deveras baixar as seguintes bibliotecas:
dlib : pip install dlib
numpy :pip install numpy
opencv  :pip install cv2
face_recognition : pip install face_recognition
